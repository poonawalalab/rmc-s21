Reviews

You must provide brief reviews of a collection (link TBD) of classic papers in robotics. These reviews must achieve three things:

  1. Describe the broad area the work fits into, and the specific problem being solved

  2. Explain the core idea of the solution

  3. Describe shortcomings, and mention papers that overcame them

Project

Simulate a robotic manipulation scenario using open source simulation methods. Your project work must contain something new, relative to what's been tried in robotics.
